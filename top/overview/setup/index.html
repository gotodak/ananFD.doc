Programmer's Reference Guide

セットアップ手順

１．動作環境

ソフトウェア 動作確認
Apache 2.2 index.html
PHP 5.3（PHP 5.4はサポート外） phpInfo.php
MySQL 5.5（phpMyAdminを含む） phpMyadmin
ZendFramework 1.10〜1.12（Ver.2はサポート外） zend.php

２．フォルダコピー

ドキュメントルートのフォルダに、ananfd、application、ananfd.doc フォルダをコピーします。

３．DB作成

phpMyAdmin のユーザ：root のパスワードを設定する。
phpMyAdmin を起動して、ユーザ：root でログインして、
　新しいユーザ gotoda【ユーザ：gotoda パスワード：gotoda】 を追加する。

ユーザ：root をログアウトして、ユーザ：gotoda でログインする。

次に、メニューからインポートを選択して、
　application/lib/data/ananfd2011.sql をアップロードファイルとして選択、インポートを実行して、
　新規データベース：ananfd2011と内部テーブルの作成済を確認する。

４．DB一括読み込み

ananfd（Webアプリ）を起動させて、マスタ設定にて各種マスタデータの一括読込を行います。

マスタ設定のユーザ管理、教務管理、学生管理、学生情報には、マスタデータが設定されていないため、
　ananfd【ユーザID：ananfd　パスワード：password】でログインして、各種マスタデータの一括読込を行います。

1) ユーザ管理

[TOP＞マスタ設定＞ユーザ管理]　において、一括読込ボタンを押下する。
[TOP＞マスタ設定＞ユーザ管理＞一括読込]　において、ファイルを選択ボタンを押下して、CSVファイルを選択する。
application/lib/data/user_2011.csv を選択して、インポートボタンを押下してインポートを実行する。

2) 教務管理

[TOP＞マスタ設定＞教務管理＞授業科目管理]　において、一括読込ボタンを押下する。
[TOP＞マスタ設定＞教務管理＞授業科目管理＞一括読込]　において、
　上から順次、ファイルを選択ボタンを押下して、CSVファイルを選択してインポートを実行する。

１．授業科目をインポートのファイルを選択ボタンを押下して、CSVファイルを選択する。
　application/lib/data/subject_2011.csv を選択して、インポートボタンを押下してインポートを実行する。

２．担当教員をインポートのファイルを選択ボタンを押下して、CSVファイルを選択する。
　application/lib/data/subteacher_2011.csv を選択して、インポートボタンを押下してインポートを実行する。

３．シラバスをインポートのファイルを選択ボタンを押下して、CSVファイルを選択する。
　application/lib/data/subfile_2011.csv を選択して、インポートボタンを押下してインポートを実行する。

3) 学生管理

[TOP＞マスタ設定＞学生管理]　において、一括読込ボタンを押下する。
[TOP＞マスタ設定＞学生管理＞一括読込]　において、ファイルを選択ボタンを押下して、CSVファイルを選択する。
application/lib/data/member_2011.csv を選択して、インポートボタンを押下してインポートを実行する。

4) 学生情報

[TOP＞マスタ設定＞学生情報＞就職先管理]　において、一括読込ボタンを押下する。
[TOP＞マスタ設定＞学生情報＞就職先管理＞一括読込]　において、ファイルを選択ボタンを押下して、CSVファイルを選択する。
application/lib/data/job_2011.csv を選択して、インポートボタンを押下してインポートを実行する。



アクセス管理設定

今回は、平成２３年度の役職一覧表を参考にして、メンバ設定を行う。

[TOP＞マスタ設定＞アクセス管理]　において、専門委員会・役職のメンバ設定を行う。

アクセス管理での各メンバは、初期値ではすべて、校長に設定されているため、
　校長【ユーザID：kocho　パスワード：kocho】でログインして、各メンバ設定を行い、
　個々のメンバ設定後、校長がメンバーで無い場合は、校長自身のメンバ削除を行う。
